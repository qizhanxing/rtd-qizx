售后服务
================

北京康吉森自动化技术股份有限公司是一家具有国际先进技术水准，融合研发、生产、销售、工程服务、售后服务于一体的高新技术企业，是中国重要的石油、化工、煤化工、天然气管线、冶金等行业安全及关键控制系统专业供应商和过程安全方案提供者。
